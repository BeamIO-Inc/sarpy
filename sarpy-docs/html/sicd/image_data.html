PixelType¶

The PixelType attribute which specifies the interpretation of the file data. Takes values in ('RE32F_IM32F', 'RE16I_IM16I', 'AMP8I_PHS8I'). Required.

Type

str

AmpTable¶

The amplitude look-up table. This is required if PixelType == ‘AMP8I_PHS8I’ Must have length exactly 256. Optional.

Type

numpy.ndarray of dtype=float64

ValidData¶

Indicates the full image includes both valid data and some zero filled pixels. Simple polygon encloses the valid data (may include some zero filled pixels for simplification). Vertices in clockwise order. Must have length >= 3. Optional.
