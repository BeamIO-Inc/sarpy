The SCPCOA object and related elements¶

The SCPCOAType definition.

class sarpy.sicd_elements.SCPCOA.SCPCOAType(SCPTime=None, ARPPos=None, ARPVel=None, ARPAcc=None, SideOfTrack=None, SlantRange=None, GroundRange=None, DopplerConeAng=None, GrazeAng=None, IncidenceAng=None, TwistAng=None, SlopeAng=None, AzimAng=None, LayoverAng=None, **kwargs)¶

Bases: sarpy.sicd_elements.base.Serializable

Center of Aperture (COA) for the Scene Center Point (SCP).

SCPTime¶

Center Of Aperture time for the SCP t_COA_SCP, relative to collection start in seconds. Required.

Type

float

ARPPos¶

Aperture position at t_COA_SCP in ECF. Required.

Type

XYZType

ARPVel¶

ARP Velocity at t_COA_SCP in ECF. Required.

Type

XYZType

ARPAcc¶

ARP Acceleration at t_COA_SCP in ECF. Required.

Type

XYZType

SideOfTrack¶

Side of track. Takes values in ('L', 'R'). Required.

Type

str

SlantRange¶

Slant range from the ARP to the SCP in meters. Required.

Type

float

GroundRange¶

Ground Range from the ARP nadir to the SCP. Distance measured along spherical earth model passing through the SCP in meters. Required.

Type

float

DopplerConeAng¶

The Doppler Cone Angle to SCP at t_COA_SCP in degrees. Required.

Type

float

GrazeAng¶

Grazing Angle between the SCP Line of Sight (LOS) and Earth Tangent Plane (ETP). Must be in the range [0.0, 90.0] Required.

Type

float

IncidenceAng¶

Incidence Angle between the SCP LOS and ETP normal. Must be in the range [0.0, 90.0] Required.

Type

float

TwistAng¶

Angle between cross range in the ETP and cross range in the slant plane. Must be in the range [-90.0, 90.0] Required.

Type

float

SlopeAng¶

Slope Angle from the ETP to the slant plane at t_COA_SCP. Must be in the range [0.0, 90.0] Required.

Type

float

AzimAng¶

Angle from north to the line from the SCP to the ARP Nadir at COA. Measured clockwise in the ETP. Must be in the range [0.0, 360.0] Required.

Type

float

LayoverAng¶

Angle from north to the layover direction in the ETP at COA. Measured clockwise in the ETP. Must be in the range [0.0, 360.0] Required.

Type

float
