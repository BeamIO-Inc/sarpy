The Earth Model. Takes values in ('WGS_84',). Default values is WGS_84. Required.

Type

str

SCP¶

The Scene Center Point (SCP) in full (global) image. This is the precise location. Required.

Type

SCPType

ImageCorners¶

The geographic image corner points array. Image corners points projected to the ground/surface level. Points may be projected to the same height as the SCP if ground/surface height data is not available. The corner positions are approximate geographic locations and not intended for analytical use. Must have length exactly 4. Required.

Type

numpy.ndarray of LatLonCornerStringType

ValidData¶

The full image array includes both valid data and some zero filled pixels. Must have length >= 3. Optional.
