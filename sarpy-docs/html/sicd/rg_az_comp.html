The RgAzComp object and related elements¶

The RgAzCompType definition.

class sarpy.sicd_elements.RgAzComp.RgAzCompType(AzSF=None, KazPoly=None, **kwargs)¶

Bases: sarpy.sicd_elements.base.Serializable

Parameters included for a Range, Doppler image.

AzSF¶

Scale factor that scales image coordinate az = ycol (meters) to a delta cosine of the Doppler Cone Angle at COA, (in 1/meter) Required.

Type

float

KazPoly¶

Polynomial function that yields azimuth spatial frequency (Kaz = Kcol) as a function of slow time (variable 1). Slow Time (sec) -> Azimuth spatial frequency (cycles/meter). Time relative to collection start. Required.

Type

Poly1DType
