EBFreqShift¶

bool: Parameter indicating whether the elctronic boresite shifts with frequency for an electronically steered array.

  • False - No shift with frequency.

  • True - Shift with frequency per ideal array theory.

    Optional.

MLFreqDilation¶

bool: Parameter indicating the mainlobe (ML) width changes with frequency.

  • False - No change with frequency.

  • True - Change with frequency per ideal array theory.

    Optional.
