The Position object and related elements¶

The PositionType definition.

class sarpy.sicd_elements.Position.PositionType(ARPPoly=None, GRPPoly=None, TxAPCPoly=None, RcvAPC=None, **kwargs)¶

Bases: sarpy.sicd_elements.base.Serializable

The details for platform and ground reference positions as a function of time since collection start.

ARPPoly¶

Aperture Reference Point (ARP) position polynomial in ECF as a function of elapsed seconds since start of collection. Required.

Type

XYZPolyType

GRPPoly¶

Ground Reference Point (GRP) position polynomial in ECF as a function of elapsed seconds since start of collection. Optional.

Type

XYZPolyType

TxAPCPoly¶

Transmit Aperture Phase Center (APC) position polynomial in ECF as a function of elapsed seconds since start of collection. Optional.

Type

XYZPolyType

RcvAPC¶

Receive Aperture Phase Center polynomials array. Each polynomial has output in ECF, and represents a function of elapsed seconds since start of collection. Optional.

Type

numpy.ndarray of XYZPolyAttributeType
