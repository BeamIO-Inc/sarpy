Type

PolarAngPoly¶

Polynomial function that yields Polar Angle (radians) as function of time relative to Collection Start. Required.

Type

Poly1DType

SpatialFreqSFPoly¶

Polynomial that yields the Spatial Frequency Scale Factor (KSF) as a function of Polar Angle. Polar Angle(radians) -> KSF (dimensionless). Used to scale RF frequency (fx, Hz) to aperture spatial frequency (Kap, cycles/m). Kap = fx x (2/c) x KSF. Kap is the effective spatial frequency in the polar aperture. Required.

Type

Poly1DType
